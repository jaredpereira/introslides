Fathom is

a scalable, trustworthy way to know what people know

technically   it's

a decentralized protocol for creating and assessing meaningful credentials through the consensus of knowledge communities.

Fathom isn't

an arbitrary credentialing standard

e.g BlockCerts, EthCerts, OpenBadges

Credential integrity is not credential vailidity

If credential consumers need to validate credentials they will gravitate towards the easiest way they can

In an institutional model this is a centralized institution

Why

Everyone should be free to build their identity however they want, and communicate it to others.

The Assessment Protocol

a.k.a the large majority of our work so far

The Components

  • Concepts
  • The Concept Tree
  • The assessment process

Concepts

A set of Identites that share a trait, skill, or quality

Each concept has parents and children

The concept tree

Each concept is a subset of it's parents.

i.e it's members are also members of parents

This forms a hierarchical ontology that defines the relationships between communities and knowledge

The Assessment Process

A sample of current members assess potential new ones with a staked schelling game

Assessee and assessors have to pay/stake with AHA (aha!) tokens

How do we get meaning from the crowd?

Assessors' inputs are weighed against each other.

A majority in consensus gets rewarded

Those not in consensus get punished

What does this give us?

A decentralized scalable mechanism for creating credentials that are:

  • directly tied to the consensus of the communities around them
  • inherently participatory, not exclusionary
  • scalable and secure

We can use these to build independant communities, systems, and structures, that can anchor identity attributes to a single source

So what does this have to do with chatrooms???

Fathom is inherently a social protocol

Most of this blockchain stuff is

Credentials are valuable because people coordinate around them in valuable ways

We need an interface that:

  • Let's us run credential communities and assessments
  • Allows us and others to create diverse, interesting, and valuable social environments around the resulting credentials

How does it work?

A smart contract defines who can be in a chatroom


Metadata defines smart contracts that users in the room can interact with

functions, events, etc

Templating and external modules connect it all and smooth the user experience

        }
